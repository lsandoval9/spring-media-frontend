Intro

This app consist in two parts: Backend and Frontend

Frontend

The frontend is made with Angular, MDBootstrap (boostrap implementation with material design), Angular material.

The Frontend has 4 entries: Home, detector, filters and about. It usage it's quite simple, filters and detect section display information with card and comunicate with the backend sending and receiving files. The home and about section contain information about the app (how it's made, how it's works).

Backend

The backend is made with spring boot and apache tika

The Backend handle the data sent from the Frontend and it processes it depending on the entry point. There are entries for file (detect section) which detects the mimetype of the file and filters (image section) which apply filters to the image depending on the entry point and some options selected by the user.
