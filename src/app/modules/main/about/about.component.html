Intro

This app consist in two parts: Backend and Frontend

Frontend

The frontend is made with Angular, MDBootstrap (boostrap implementation with material design), Angular material.

The Frontend has 4 entries: Home, detector, filters and about. It usage it's quite simple, it display information with card and then comunicate with the backend.

Backend

The backend is made with spring boot and apache tika

The Backend handle the data send from the Frontend and apply filters or return the mimetype of the file
