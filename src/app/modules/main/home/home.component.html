Intro

Hello, my name is Luis Augusto and this is my first full stack application. The application consists of tools to apply filters to images and to detect the mimetype of a file. The app is quite simple, it consist in 2 sections:

  • Mimetypes
  • Filters

the filter section was made with the same filters from the CS50X course and some others that I added like negative, contrast, saturation (hue).

The detector section was made using Apache Tika, a tool for manipulate and detect files metadata

MIME types

Here is the definition of MIME type from the MDN foundation

In the Detect section you can detect the MIME type of a provided file, this is useful when you have a file without a valid extension as ".jpg", ".pdf", etc. and you want to know which type of file it is.

The mimetypes generally consists in and type and a subtype divided with a slash (/). The type represent a general category that indicates the file's purpose such a video, text, image, audio. The subtype indicates the exact kind of data used by the file, some examples may be mp4, plaintext, jpg, mp3.

Here is a valid mimetype example

image/jpeg
